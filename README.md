Front end challenge for BZ.

### 1. Install dependencies

```sh
$ npm install
```

### 2. Run the app

```sh
$ npm start
```

Should open [http://localhost:3000](http://localhost:3000) in browser 


### Run Storybook

```sh
$ npm run storybook
```
Should open [http://localhost:9000](http://localhost:9000) in browser 

### Run Tests

```sh
$ npm test
```
